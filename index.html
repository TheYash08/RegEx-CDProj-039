PHASES
Five Compiler Phases

Each phase maps directly to a core compiler design concept.

PHASE 01
🔤
Lexer — Tokenization
Reads the regex pattern character by character. Classifies each as CHAR, STAR, PLUS, QUES, OR, LPAREN, RPAREN, or DOT.
LEXICAL ANALYSIS
PHASE 02
🌳
Parser — AST
Recursive Descent Parser builds an Abstract Syntax Tree. Handles operator precedence: quantifiers > concatenation > alternation.
RECURSIVE DESCENT
PHASE 03
🔀
NFA — Thompson's
Thompson's Construction converts each AST node into an NFA fragment via ε-transitions. Produces single start and accept state.
THOMPSON'S ALGO
PHASE 04
📊
DFA — Subset Construction
Converts NFA to a Deterministic Finite Automaton. Computes ε-closures for every reachable state set using powerset construction.
SUBSET CONSTRUCTION
PHASE 05
✂️
DFA Minimization
Hopcroft's Algorithm merges equivalent states by partitioning them into distinguishable groups. Produces smallest possible DFA.
HOPCROFT'S ALGO
BONUS
⚡
Visual Graphs
NFA, DFA and Minimized DFA are drawn as state diagrams — circles, arrows, accept states — so your professor can verify correctness.
